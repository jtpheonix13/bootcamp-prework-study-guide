CSS

  • Inline Css is done directly in the HTML File
  • Internal CSS style sheets are done in the HTML file for specific elements
  • External CSS style sheets keep all the CSS rules in a separate file

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone, create branch, edit, git add command, git commit, git pull, git push origin new branch
  • git add -A command commits all files in the working branch.
  • git commit -m to add message to our commit in quotes
  • git pull origin main pulls base branch into working branch

JavaScript

  • Javascript adds functionality to pages by using logic statements
  • Conditional for and if statements check to see if certain statements exist or are true and carry out actions based on the results
  • A variable is a named container to store information
  • Control flow is the order in which a computer executes code in a script
